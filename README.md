## Dotfiles

必备的配置文件要保留起来便于迁移，一定别忘记备份、迁移配置文件。

- vim: ~/.vim/
- oh-my-zsh: ~/.zshrc
- ssh:  ~/.ssh/...
 包括config, known_hosts, 私钥, 公钥等文件，注意公钥文件的权限要是400。
- git: ~/.gitconfig
- Alfred: /Users/`MyName`/Library/Application Support/Alfred 2/...
